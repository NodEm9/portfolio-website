Meet App

This Meet App is a serverless, progressive web application (PWA) that uses the Google Calendar API to fetch upcoming events. The app allows users to filter events by city, show/hide event details, specify the number of events they want to view, and add an event to their Google Calendar. The app is built with React and uses a test-driven development (TDD)and (BDD) approach. All tests are written using Jest, Ghekins, Puppeteer and Jest-Cucumber. Atatus was used to monitor the performance of the app and Recharts to display data visualixation of the events of respective cities.
